Ejemplo de CSS

Este es un ejemplo de un contenedor con margen, padding y borde.

Explicación:

Margin: 50px de margen alrededor del contenedor.

Padding: 20px de espacio interior entre el contenido y el borde.

Border: Borde de 5px sólido y verde alrededor del contenedor.

Width & Height: Ancho de 300px y alto de 200px para el contenedor.
