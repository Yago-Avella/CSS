Ejemplo de Marcos CSS

Explicación:

Margin El margen es el espacio exterior de un elemento que lo separa de otros elementos en la página. Es como el espacio que rodea un bloque o caja. No afecta el tamaño del contenido del elemento. Se puede ajustar para cada lado (arriba, derecha, abajo, izquierda) o aplicar un valor global que afecte a todos..

Padding El relleno (padding) es el espacio interior entre el contenido de un elemento y su borde. Es como un acolchado que aleja el contenido de los bordes del propio elemento. A diferencia del margin, el padding sí afecta el tamaño visual del elemento, ya que expande el área interna.

Border El borde (border) es el contorno visible que rodea un elemento. Puedes ajustar el grosor, estilo y color del borde. Se coloca entre el padding y el margin, por lo que también afecta el tamaño visual del elemento.

Width y HeightEl borde (border)width (ancho) y height (alto) son las dimensiones del área del contenido del elemento. Definen el espacio que ocupa el contenido sin contar el padding, border o margin. Puedes usar diferentes unidades como píxeles (px), porcentajes (%), y unidades relativas como vw (ancho de la ventana).
