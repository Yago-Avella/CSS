HTML

Las clases (class) y los identificadores (id) en CSS se diferencian en su uso y especificidad.

Un ID es único y se aplica a un solo elemento, ideal para estilos o interacciones específicas.

Una clase puede reutilizarse en varios elementos, útil para aplicar estilos generales a grupos con características similares.


CSS

Se aplican estilos al título usando #titulo.

Los tres párrafos comparten el estilo de la clase .texto.

El botón tiene un estilo específico, combinando la clase .boton y el id #botonEnviar. Además, hay un estilo adicional cuando se pasa el mouse sobre él (hover).
