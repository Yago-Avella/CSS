Este es el título

HTML

El título tiene un id #titulo.

Los párrafos comparten una clase .texto.

El botón tiene tanto la clase .boton como el id #botonEnviar.


CSS

Se aplican estilos al título usando #titulo.

Los tres párrafos comparten el estilo de la clase .texto.

El botón tiene un estilo específico, combinando la clase .boton y el id #botonEnviar. Además, hay un estilo adicional cuando se pasa el mouse sobre él (hover).
